# Dr. C. Ramachandran — Academic Portfolio Website

iPortfolio-based GitHub Pages site styled after [Narayana-Rao/Narayana-rao.github.io](https://github.com/Narayana-Rao/Narayana-rao.github.io).

## Sections
- **Home** — particles.js hero, name, tagline, social links
- **About** — bio, contact details, profile photo
- **Awards / Fellowships** — SRF, URF, CSIR-UGC NET, TNSET, best oral presentation
- **Training & Workshops** — 6 tiles with certificate/photo zoom
- **Resume** — education (Ph.D., M.Phil., M.Sc.), career (Guest Lecturer, Govt. Arts College Salem), specializations, instrumentation, software, memberships, languages
- **Research** — 13 journal articles + 6 conference presentations with Copy BibTeX / Copy Harvard buttons, filterable (All / Journals / Conferences)
- **Gallery** — Spotlight lightbox grid (grayscale → colour on hover)
- **Contact** — address, emails, phones

## Photos to Replace

## Deploy on GitHub Pages
1. Create a repo named `<your-username>.github.io`
2. Push this entire folder to the repo
3. Enable GitHub Pages (Settings → Pages → Source: main branch, root)
4. Site will be live at `https://<your-username>.github.io`
